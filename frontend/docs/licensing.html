The MSP plan is built for managed-service providers running tenants for multiple client organizations: group tenants into client orgs, give each client scoped org_admin / org_viewer accounts, and track billing memos and renewal dates per client (see the MSP & client orgs page). Licenses are sold at flat, published prices - no per-end-user counting, no sales calls. Purchase and upgrades: idpvault.com.

How licensing works

When you purchase, you receive a license key (IDPV-XXXX-XXXX-XXXX-XXXX). Paste it in Administration → License: the app exchanges it with the KelTech license server (license.keltech.ai) for a cryptographically signed entitlement bound to this install, then verifies that entitlement offline against a public key built into the app. The only data ever sent is the license key itself and a random install id - never your tenants, never your configuration, never any usage data. There is no telemetry anywhere in the product.

One license runs on one install at a time. Moving to a new install is self-service: click Deactivate license on the old install (or release it from the customer portal at idpvault.com/manage), then activate the same key on the new one. A daily background check-in picks up renewals and tenant add-ons automatically - renewing never requires touching the app.

Air-gapped and offline installs

Annual licenses can run with zero network access: the customer portal issues an offline license file for your install id (shown on the License page). Upload it in Administration → License - it is verified entirely offline and the app never contacts any server. Offline files are issued per paid term, so renew and download a fresh file before expiry. The free Community tier also never makes any network connection.

Terms & renewal

Licenses are sold as annual terms. Activated installs pick up renewals automatically at the next daily check-in. Offline installs download a fresh license file from the portal after renewing. The License panel always shows the expiry date and a day countdown. Tenant add-ons ($500/tenant/yr, Business and MSP) can be purchased whenever needed and apply at the next check-in (or via a fresh offline file).

If a license expires or is removed

After a 3-day grace window, the instance downgrades non-destructively: nothing is deleted. Your oldest tenant stays fully operational; other tenants keep all data and snapshots but pause backup/restore; Users & Access features pause. Everything resumes the moment a valid key is installed.
